Rules

Add to Cart Button (Wishlist)

  • Hover ON
    • Cursor changes to pointer
    • "Add to Cart" button opacity increases to 100%
    • Button background gradient animates in seamless loop
  • Hover OFF (Add to Cart Default)
    • Cursor changes back into arrow
    • Button opacity decreases back to 75%
    • Button background gradient becomes static
  • On Click
    • Overlay background appears at 85% opacity
    • "Added to your cart!" confirmation modal eases upwards for 300 milliseconds
    • Set the user's cart flag to TRUE in the database
    • Update the user's cart item count to 1 in the database

Added Modal (Close Button)

  • Hover ON
    • Cursor changes to pointer
    • Close button color changes to black
  • Hover OFF (Default)
    • Cursor changes back into arrow
    • Close button color changes to #aaa
  • On Click
    • "Added to your cart!" confirmation modal eases downwards for 300 milliseconds
