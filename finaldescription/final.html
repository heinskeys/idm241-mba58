Rules

Game Rank Field

  • Hover ON
    • Bacground increases in opacity to 100%
    • Cursor change into move icon
    • Tooltip appears
  • Hover OFF
    • Background decreases in opacity to 75%
    • Cursor changes back into arrow
    • Tooltip disappears
  • On Click
    • Game Rank Field becomes editable
    • Game Rank Field border changes to blue
    • Game Rank Field text color changes to black
    • Tooltip disappears

Tag Buttons

  • Hover ON
    • Cursor changes to pointer
    • Tag button size increase by 15%
    • Tag button background color changes to #333
  • Hover OFF
    • Cursor changes back into arrow
    • Tag button size decreases by 15%
    • Tag button background color changes back to #222
  • On Click
    • Tag category page opens in new tab

Add to Cart Button (Wishlist)

  • Hover ON
    • Cursor changes to pointer
    • "Add to Cart" button opacity increases to 100%
    • Button background gradient animates in seamless loop
  • Hover OFF (Add to Cart Default)
    • Cursor changes back into arrow
    • Button opacity decreases back to 75%
    • Button background gradient becomes static
  • On Click
    • Overlay background appears at 85% opacity
    • "Added to your cart!" confirmation modal eases upwards for 300 milliseconds
    • Set the user's cart flag to TRUE in the database
    • Update the user's cart item count to 1 in the database

Added Modal (Close Button)

  • Hover ON
    • Cursor changes to pointer
    • Close button color changes to black
  • Hover OFF (Default)
    • Cursor changes back into arrow
    • Close button color changes to #aaa
  • On Click
    • "Added to your cart!" confirmation modal eases downwards for 300 milliseconds

View Cart Button

  • Hover ON
    • Cursor changes to pointer
    • "View Cart" button opacity increases to 100%
    • Button background gradient animates in seamless loop
  • Hover OFF
    • Cursor changes to pointer
    • "View Cart" button opacity decreases back to 75%
    • Button background gradient becomes static
  • Mouse Down
    • Button increases in size by 8%
  • Mouse Up
    • Button decreases in size by 8%
    • User cart pages opens up in new tab
