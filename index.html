ALPHA DESCRIPTION

https://store.steampowered.com/

Triggers

  • Hover ON
  • Hover OFF
  • On Click

Rules

  • Hover ON
    • Cursor changes to finger
    • Button opacity increases to 100%
    • Rank/order input field becomes editable
    • Rank/order input field background changes to #313c48 with semi-transparent shadow
  • Hover OFF
    • Cursor changes back into pointer
    • Button opacity decreases back to 75%
    • Rank/order input field cannot be edited (default)
    • Rank/order input field background changes to rgba(64, 81, 99, 0.9) with no shadow (default)
  • On Click
    • Overlay background appears at 85% opacity
    • "Added to your cart!" confirmation modal appears
    • Rank/order input field cannot be edited (default)
    • Rank/order input field background changes to rgba(64, 81, 99, 0.9) with no shadow (default)
    • Database user cart boolean set to TRUE
